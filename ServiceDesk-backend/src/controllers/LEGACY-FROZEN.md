# Legacy Controllers — Frozen

This directory is **frozen** as of 2026-03-20.

## Allowed Changes

- Bug fixes
- Security patches
- Compatibility fixes for existing consumers

## Forbidden

- New controller files
- New handler functions in existing files
- New business logic

## Where to Put New Code

All new controllers must be implemented in `src/modules/*/controllers/`.

See [Architecture Policy](../../../docs/architecture/architecture-policy.md) for details.
